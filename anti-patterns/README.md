## 反例格式说明

每个反例都包含：

- ❌ **错误示例**：展示常见的错误代码
- 🚨 **问题说明**：解释为什么这是错误的
- ✅ **正确做法**：提供正确的实现方案
- 💡 **最佳实践**：额外的建议和技巧

## 使用指南

1. **浏览目录**：根据你的问题找到相关反例
2. **对比学习**：理解错误与正确的区别
3. **应用到代码**：修改你的实际代码
4. **定期回顾**：避免重复犯错

## 贡献指南

如果你发现了新的反例，欢迎贡献：

1. 创建新文件或添加到现有文件
2. 遵循格式说明
3. 提供清晰的问题解释
4. 包含可运行的代码示例

## 记住
